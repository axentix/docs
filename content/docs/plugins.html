This page lists all the different plugins you can add to your projects.

A plugin is an addon made by the community and verified by the Axentix Team to ensure it has the same behavior as an Axentix component.
You can add any plugin inside your project to get new features who's not already inside Axentix. Github Repository

Create your plugin

You can create your own plugin on the plugin github by following the CONTRIBUTING instructions.
