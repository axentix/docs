--- title: 'Select' description: 'Axentix select. The select allows you to create an input which contains different options.' type: 'docs' sidenav: forms ---

Basic select

Create a basic select by adding the .form-control class on it.
The .disabled class or attribute works on the select too.

Multiple select

Creating a multiple select is really simple, you just have to add the multiple attribute on the select element.
