File input

The file input makes the user able to send a file inside a form.
Selecting multiple files inside your form will make their name appear inside the .form-file-path div separated with comas.

Methods

Method Description
Axentix.Forms.updateInputs() This method is scanning the HTML page to update all the file inputs & material forms.
