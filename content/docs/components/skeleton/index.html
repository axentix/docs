Skeleton

To create a skeleton loading item, use the class="skeleton" class.

Gradient skeleton

For a gradient animation of the skeleton loading, use the class="skeleton-gradient" class.

Styling

CSS Variable Default value Description
--ax-skeleton-background-from #e0e4e7 Default starting color of the skeleton animation.
Accepts all css color values (rgba, hex, rgb).
--ax-skeleton-background-to #f0f3f5 Default ending color of the skeleton animation.
Accepts all css color values (rgba, hex, rgb).
