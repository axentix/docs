Icons use

We recommend to use the Iconify library to create icons.
This library contains all the common icon libraries and allows you to get any of them using the right syntax.

Select the icon you want to use and use the class="iconify-inline" class the data-icon="library:iconName" syntax.
This documentation uses the Material Design Icons (MDI) icon set.
